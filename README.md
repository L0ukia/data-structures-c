# Data Structures in C

Implementations of fundamental data structures in C,
developed as part of my Data Structures course.

## Contents

### Stack
| File | Description |
|------|-------------|
| stack_array_int.c | Integer stack with Push/Pop operations |
| stack_array_char.c | Character stack used for palindrome checking |
| stack_array_struct.c | Stack storing structured data (price/item records) |
| stack_linked_list.c | Dynamic stack using linked list nodes |

### Queue
| File | Description |
|------|-------------|
| queue_array_stats.c | Integer queue with statistical operations |
| queue_array_even_odd.c | Splits integers into Even/Odd queues |
| queue_array_search.c | Queue with linear search |
| queue_linked_list.c | Dynamic queue using linked list nodes |

### Linked List
| File | Description |
|------|-------------|
| list_basic.c | Insert, traverse, search on a singly linked list |
| list_merge.c | Merges two linked lists into one |
| list_delete.c | Insert and delete nodes |
| list_search.c | Search across two linked lists |
| list_with_stack.c | Combined list and stack operations |

## Language
C (C99 standard)
